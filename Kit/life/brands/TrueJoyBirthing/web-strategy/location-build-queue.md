## Selection Criteria

Cities must meet ALL of:
1. **Population**: 50k–250k or clear outer-ring suburb
2. **Growth**: Fast-growing, family-heavy
3. **SERP winnable**: Page 1 mostly individuals, small groups, directories — no wall of huge national brands
4. **Vibe**: Natural/holistic-friendly — birth centers nearby, midwife practices, community support
5. **Geographic spread**: No more than 2 cities per state in a single batch

## Rules

- **Review branches only.** Never auto-deploy.
- **Max 2-3 deploys per 24h.**
- **Explicit human approval before every deploy.**
- **Similarity gate**: If D1 hospital-paragraph similarity > 65% vs any existing city, flag before including in review packet.
- **All new cities must pass validation** (`validate-cities.ts`, `city-pages.test.ts`) before review packet delivery.
- **NICU claims**: Use "verified Level X NICU" with source, or "contact the hospital directly" if unverifiable. No bare "Level X NICU" claims.
- **Pre-send cross-check**: Verify all factual claims in outreach drafts match live page content before sending.
